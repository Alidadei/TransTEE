# TransTEE 环境配置说明

## 概述

TransTEE 项目包含四个子模块，每个模块有不同的依赖需求。本文档说明如何使用 Conda 配置统一的 GPU 运行环境。

### 项目结构

```
TransTEE/
├── CausalM/          # 语言模型因果研究 (BERT, NLP)
├── Continuous/       # 连续治疗效应估计
├── Dosage/           # 剂量治疗效应估计
├── Structured/       # 图结构数据处理
├── docs/             # 技术文档和经验总结
├── record/           # 实验结果记录
└── tests/            # 测试和验证代码
```

## 环境配置

### 1. 创建 Conda 环境

使用项目根目录下的 `transtee_gpu_env.yml` 配置文件：

```bash
# 进入项目目录
cd B:/Experiments/TransTEE

# 创建环境 (使用本地配置文件，环境安装在项目目录下)
conda env create -f transtee_gpu_env.yml

# 激活环境
conda activate transtee
```

### 2. 验证 GPU 支持

```bash
# 检查 PyTorch 是否识别 GPU
python -c "import torch; print(f'CUDA available: {torch.cuda.is_available()}'); print(f'CUDA version: {torch.version.cuda}')"

# 检查 PyTorch Geometric (用于 Structured 模块)
python -c "import torch_geometric; print(f'PyG version: {torch_geometric.__version__}')"
```

### 3. 下载 NLP 模型 (CausalM 模块)

```bash
# 下载 spaCy 英文模型
python -m spacy download en_core_web_lg

# 下载 NLTK 数据
python -c "import nltk; nltk.download('punkt'); nltk.download('stopwords')"
```

## 常见问题

### CUDA 版本不匹配

如果遇到 CUDA 版本问题，检查 NVIDIA 驱动版本：

```bash
nvidia-smi
```

根据驱动版本调整 `transtee_gpu_env.yml` 中的 `pytorch-cuda` 版本。

### PyTorch Geometric 安装失败

PyG 需要与 PyTorch 版本匹配。如果自动安装失败，手动安装：

```bash
pip install torch-geometric
pip install torch-scatter torch-sparse -f https://data.pyg.org/whl/torch-2.1.0+cu121.html
```

### 内存不足

训练大型模型时，减少 batch size 或使用梯度累积：

```python
# 在训练脚本中添加
optimizer.zero_grad()
for i, batch in enumerate(dataloader):
    loss = model(batch)
    loss.backward()
    if (i + 1) % accumulation_steps == 0:
        optimizer.step()
        optimizer.zero_grad()
```

## 实验运行指南

### CausalM 模块

```bash
cd CausalM/POMS_GendeRace/pipeline
python training.py --config config.yaml
```

### Structured 模块

```bash
cd Structured
python main.py --dataset <dataset_name>
```

### Continuous 模块

```bash
cd Continuous
python main_batch.py --data_dir ./dataset/ihdp/continuous/0
```

### Dosage 模块

```bash
cd Dosage
python test_TransTEE.py
```
