# TransTEE 实验运行指南

## 概述

TransTEE 项目包含四个子模块，每个模块针对不同类型的治疗效应估计问题：

| 子模块 | 问题类型 | 数据类型 | 主要脚本 |
|--------|----------|----------|----------|
| **CausalM** | 离散治疗（性别/种族） | 文本 (POMS) | `training.py` |
| **Continuous** | 连续治疗 | 表格数据 (IHDP/News) | `main_batch.py` |
| **Dosage** | 剂量治疗 | 分子数据 (TCGA) | `test_TransTEE.py` |
| **Structured** | 结构化治疗 | 图数据 (QM9/TCGA) | `run_model_training.py` |

---

## 1. CausalM 模块 - 语言模型因果研究

### 数据准备

```bash
# 1. 从 Kaggle 下载数据集
# https://www.kaggle.com/amirfeder/causalm
# 下载后解压到 CausalM/POMS_GendeRace/datasets/

# 2. 设置数据路径
# 编辑 CausalM/POMS_GendeRace/pipeline/constants.py
# 修改 CAUSALM_DIR 变量为数据集路径
```

### 训练与测试

```bash
cd CausalM/POMS_GendeRace/pipeline

# 性别治疗效应估计
python training.py --treatment gender --model transtee

# 种族治疗效应估计
python training.py --treatment race --model transtee

# 其他基线模型
python training.py --treatment gender --model drnet
python training.py --treatment gender --model tarnet
python training.py --treatment gender --model vcnet
```

### 输出

- 训练日志保存在 `logs/` 目录
- 评估指标：PEHE (Precision in Estimation of Heterogeneous Effect)

## 2. Continuous 模块 - 连续治疗效应估计

### 数据准备

```bash
cd Continuous

# 生成合成数据集 (Simu1)
python data/simu1_generate_data.py

# 生成 IHDP 数据集
python data/ihdp_generate_data.py

# 生成 News 数据集
python data/news_process.py
python data/news_generate_data.py
```

### 训练与测试

#### Simu1 数据集

```bash
# 单次运行 + 绘制 ADRF 曲线
python main_batch.py --plt_adrf

# 批量运行 (100个数据集)
python main_batch.py

# 使用 TR 正则化
python main_batch.py --p 1

# 使用 PTR 正则化
python main_batch.py --p 2
```

#### IHDP 数据集

```bash
# 单次运行
python main_batch_ihdpv2.py --plt_adrf

# 批量运行
python main_batch_ihdpv2.py

# 使用正则化
python main_batch_ihdpv2.py --p 1  # TR
python main_batch_ihdpv2.py --p 2  # PTR
```

#### News 数据集

```bash
# 单次运行
python main_batch_newsv2.py --plt_adrf

# 批量运行
python main_batch_newsv2.py
```

### 输出

- 评估指标：MISE (Mean Integrated Squared Error)
- ADRF 曲线图保存在相应目录

## 3. Dosage 模块 - 剂量治疗效应估计

### 数据准备

```bash
cd Dosage

# 下载 TCGA 数据集
# https://drive.google.com/file/d/1P-smWytRNuQFjqR403IkJb17CXU6JOM7/view?usp=sharing
# 解压到 datasets/ 目录
```

### 训练与测试

#### TransTEE 模型

```bash
# 基础 TransTEE
python test_TransTEE.py \
    --num_treatments=3 \
    --treatment_selection_bias=2.0 \
    --dosage_selection_bias=2.0 \
    --batch_size=128 \
    --h_dim=64 \
    --h_inv_eqv_dim=64 \
    --num_dosage_samples=5 \
    --alpha=1.0 \
    --model_name="TransTEE"

# TransTEE + TR 正则化
python test_TransTEE.py --p=1 --model_name="TransTEE_TR"

# TransTEE + PTR 正则化
python test_TransTEE.py --p=2 --model_name="TransTEE_PTR"
```

#### 基线模型

```bash
# SCIGAN 基线
python test_SCIGAN.py \
    --num_treatments=3 \
    --treatment_selection_bias=2.0 \
    --dosage_selection_bias=2.0 \
    --batch_size=128 \
    --h_dim=64 \
    --model_name="scigan"

# DRNet, TarNet, VCNet 基线
python test_DRNet.py \
    --num_treatments=3 \
    --treatment_selection_bias=2.0 \
    --dosage_selection_bias=2.0 \
    --model_name=['drnet', 'tarnet', 'vcnet']
```

### 输出

- 评估指标：MISE, DPE (Dosage Policy Error), PE (Policy Error)
- 模型保存在 `saved_models/` 目录

## 4. Structured 模块 - 图结构治疗效应估计

### 数据准备

```bash
cd Structured

# 数据会自动下载，或手动下载：
# TCGA: https://drive.google.com/file/d/1P-smWytRNuQFjqR403IkJb17CXU6JOM7/view?usp=sharing
# QM9: http://deepchem.io.s3-website-us-west-1.amazonaws.com/datasets/gdb9.tar.gz
# 放置到 data/tcga/ 目录
```

### 生成数据

```bash
# SW 模拟数据
python generate_data.py --task sw --bias 1.0 --seed 0

# TCGA 模拟数据
python generate_data.py --task tcga --bias 1.0 --seed 0
```

### 训练与测试

```bash
# TransTEE 模型
python run_model_training.py \
    --task tcga \
    --model transtee \
    --bias 1.0 \
    --seed 0

# 其他模型
python run_model_training.py --task tcga --model sin      # SIN
python run_model_training.py --task tcga --model gnn      # GNN
python run_model_training.py --task tcga --model cat      # CAT
python run_model_training.py --task tcga --model graphite # Graphite

# 超参数搜索
python run_hyperparameter_sweeping.py
```

### 输出

- 评估指标：PEHE
- 结果保存在 `results/` 目录
- 可选同步到 wandb.ai

## 常见问题

### 1. 数据集未找到

确保数据已下载并放置在正确的目录：
- CausalM: `datasets/` 文件夹
- Dosage/Structured: `datasets/` 或 `data/tcga/` 文件夹

### 2. CUDA 内存不足

减少 batch size：
```bash
python test_TransTEE.py --batch_size=64  # 默认 128
```

### 3. 训练时间过长

- 减少训练轮数：`--max_epochs 100` (默认 200/800)
- 减少数据集数量：`--num_dataset 10` (默认 100)
